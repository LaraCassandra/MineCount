About MineCount

MineCount is a Minecraft Inventory Tracking application made using C#. This project was made for Interactive Development 300 in term 2. We were required to create an interactive web application that acts as a Minecraft Inventory tracker. The user is able to track their Minecraft blocks and store the count in a database and craft recipes from those blocks.
